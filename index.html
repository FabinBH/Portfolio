Principais Projetos

Cadastro de alunos

O projeto é um sistema de cadastro de alunos onde os usuários se registram por e-mail e podem fazer uma prova para obter um certificado, sendo permitido apenas uma tentativa por aluno. Desenvolvido em Java com o Spring Framework, o sistema utiliza endpoints RESTful para manipulação de dados e gerenciamento de sessões, garantindo eficiência e escalabilidade no desenvolvimento web.
